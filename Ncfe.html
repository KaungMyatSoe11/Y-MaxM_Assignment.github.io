Diploma in Cloud Computing (DCC)

"Diploma in Cloud Computing (DCC)" သင်တန်းကို Y-Max University တွင်ဖွင့်လှစ်ပါမည်။

🎯ဘယ်သူတွေတက်ရောက်နိုင်သလဲ?

👉Cloud Computing ပိုင်းကို ယခုမှစတင်လေ့လာ လိုသူများ၊
👉Cloud Infrastructure/ AWS Cloud ပိုင်းကို စိတ်ပါဝင်စားသူများ၊
👉ကွန်ပျူတာတက္ကသိုလ်(CU) ၊ နည်းပညာတက္ကသိုလ် (TU)မှ ကျောင်းသား ၊ ကျောင်းသူများ၊
👉Core Skill တွေဖြစ်တဲ့ Microsoft Server / Network Administration ပိုင်းတွင်အခြေခံ ရှိသူများ၊
👉မိမိ Career ကိုမြှင့်တင်လိုပြီး လစာပိုမိုကောင်းမွန် လိုသူများ တက်ရောက်နိုင်ပါတယ်။

🔴သင်တန်းကာလပြီးဆုံးပါက Cloud Infrastructure ကို မိမိကိုယ်တိုင် တည်ဆောက်နိုင်ရုံအပြင် Exam ဖြေဆို ‌အောင်မြင်ပါက UK က အသိအမှတ်ပြုပေးအပ်တဲ့ Diploma ဘွဲ့ကို ရရှိမှာဖြစ်ပါတယ်။

🔴အဓိက ကတော့ UK က Professional Diploma တွေကို ပြည်တွင်းမှာပဲ စျေးနှုန်းချိုသာစွာနဲ့ UK Standard အတိုင်း သင်ကြားပေးပြီး UK ကဘွဲ့ကို အချိန်တိုအတွင်းရယူလိုတဲ့ သူတွေ တက်ရောက်သင့်ပါတယ်။
🎯"Diploma in Cloud Computing" သင်တန်းတွင် သင်ကြားပေးမည့် Main Topics များ
📕Introduction to Cloud Computing
📕Cloud Service Model
📕Components of Cloud Computing
📕Introduction to AWS Cloud Infrastructure
📕Administration of AWS Cloud Service (EC2,S3, Glacier, Elastic_IP,EBS, Load_Banlancer, Work Mail, Route_53,RDS and more lab. )
📕Introduction to AZURE Cloud Infrastructure
📕Core Service Configuration of AZURE

Diploma in IT Management (DITM)

     "Diploma in IT Management (DITM)" သင်တန်းကို Y-Max University တွင်ဖွင့်လှစ်ပါမည်။

🎯ဘယ်သူတွေတက်ရောက်နိုင်သလဲ?

👉IT ပညာရပ်နှင့်ပတ်သက်သောစီမံခန့်ခွဲမှုကို လေ့လာလိုသူများ၊
👉လုပ်ငန်းခွင်တွင် ကွန်ပြူတာနှင့် ပတ်သက်သော Hardware၊ Software များကို ကိုင်တွယ်ဖြေရှင်းရုံ သာမက နည်းပညာဆိုင်ရာ ပြဿနာများ၊IT Operations များ၊ Database Security များကို ကောင်းမွန်စွာ စီမံခန့်ခွဲနိုင်လိုသောသူများ၊
👉လုပ်ငန်းခွင်တွင် IT Manager အဖြစ်နောက်ထပ် တစ်ဆင့်သို့တက်လှမ်းလိုသူများ၊
👉IT manager တစ်ယောက်အနေဖြင့် သိသင့်သိထိုက် သော လုပ်ငန်းဆောင်တာများ၊
👉Network ချိတ်ဆက်မှု၊ IT Project များတွင် စီမံခန့်ခွဲမှုလုပ်ငန်းစဉ်များကို လေ့လာလိုသောသူများ၊
👉မိမိ Careerကိုမြှင့်တင်ပြီးလစာပိုမိုကောင်းမွန်လိုသူ များ တက်ရောက်နိုင်ပါသည်။

🔴အဓိက ကတော့ UK က Professional Diploma တွေကို ပြည်တွင်းမှာပဲ စျေးနှုန်းချိုသာစွာနဲ့ UK Standard အတိုင်း သင်ကြားပေးပြီး UK ကဘွဲ့ကို အချိန်တိုအတွင်းရယူလိုတဲ့ သူတွေ တက်ရောက် သင့်ပါတယ်။

🎯"Diploma in IT Management (DITM)" သင်တန်းတွင် သင်ကြားပေးမည့် Main Topics များ

📕IT System management and Operational Strategies
📕Role of Manager in Information Technology
📕Skills required for IT Manager
📕Business Functions of IT issues
📕Management Issues and Security Issues
📕Operation System Concepts
📕Mobile Technology
📕Understanding Software and Database Technologies
📕Overview of Project Planning
📕Project Development Lifecycle
